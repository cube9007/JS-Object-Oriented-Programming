04_1강 객체는 언제 쓰나요?

정리 정돈 도구

파일을 정리할 때 폴더(디렉토리)가 없으면 정리가 매우 힘들 것임
코딩도 마찬가지, 함수나 변수가 너무 많으면 이름이 길어지고 정리하기가 어려워짐
함수와 변수를 폴더 안에 넣는 것처럼 객체 안에 넣는 것임

Math객체 예시

document.write(Math.floor( 10 * Math.random() ) + ' : 랜덤 숫자 <br>'); document.write(Math.floor( 10 * Math.random() ) + ' : 랜덤 숫자 <br>'); 
Math라는 객체 안에다가 random()이란 메소드를 정리해놓은 것임
