객체 만들어보기

MyMath라는 객체를 만들어 보기
var MyMath = { PI : Math.PI , random : function() { return Math.floor(Math.random() * 10); }, floor : function(value) { return Math.floor(value); } } document.write( MyMath.PI + '<br>'); document.write( MyMath.random() + '<br>'); document.write( MyMath.floor(Math.random() * 100) + '<br>'); 
객체라는 수납상자에 Math와 관련된 변수, 함수들을 넣어놓음
