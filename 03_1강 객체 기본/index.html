가벼운 몸풀기 익숙한 배열 ^^

배열로 몸풀기

<script> var baeyeol = ['wonho','yuri','taegook']; document.write('baeyeol[1]' + baeyeol[1]); </script>

객체로 몸풀기

<script> var gaekchae = { designer : 'wonho', planner : 'yuri', jjang : 'taegook', } document.write( 'gaekchae.planner : ' + gaekchae.planner ) </script>

객체의 기본

객체는 {중괄호}를 사용함
객체를 불러올 때는 객체이름.메소드 - 점.으로 불러올 수도 있지만
              객체이름['메소드'] - ['대괄호와 따옴표']를 사용해서 불러올 수도 있음
객체를 추가 수정 할 떄는 객체이름.메소드 = '수정사항' 으로 수정할 수 있음
객체를 삭제할 떄는 delete 객체이름.메소드 로 삭제할 수 있음

객체 {중괄호} 사용

var gaekchae = { designer : 'wonho', plaaner : 'yuri', }

객체 불러오기

gaekchae.designer gaekchae['designer']

객체 추가 수정

gaekchae.designer = 'daegeun';

객체 삭제

delete gaekchae.designer
