프로그램에서도 자기 자신을 가르키는 표현이 있음 = this
굉장히 중요 !!
var kim = { name : kim, first : 10, second : 20, sum : function() { return this.first + this.second; } } document.write('kim.sum() : ' + kim.sum()); 
this 는 자기가 속해있는 객체를 가르킴
this를 쓰면 객체의 이름이 바뀌어도 자기 자신을 가르키는 건 변함 없기 때문에, 자기 자신을 가르키는 코드를 바꿀 필요가 없음
