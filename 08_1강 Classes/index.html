08_1강 Classes

클래스의 의미 

자바스크립트는 전통적인 객체지향의 문법을 채택함 - 이미 객체 지향을 사용하는 사람들이 문법적인 거부감 없이 자바스크립트를 사용할 수 있게 함
그런 문법 중에 하나가 class
자바, 파이선, php 등에서도 쓰임
자바스크립트는 지원 안했는데 constructor function() 으로 객체를 찍어내고 있었음 
근데 class라는 문법을 도입하면서 기존 객체 지향 문법을 아는 사람이 쉽게 배울 수 있게 되고
자바스크립트의 class를 배운 사람은 다른 컴퓨터 언어로 갔을 때도 편함 

class = 객체를 찍어내는 공장 , constructor function() 의 대체제 

자바스크립트 es6이라는 표준이 있음 6버전에서 추가된 기능 - class
6 이전 버전에선 동작하지 않음
그래도 문제되지 않음, 요즘은 많은 언어들이 6버전을 지원함
